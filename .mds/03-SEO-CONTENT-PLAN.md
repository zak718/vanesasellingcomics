## On-Page SEO Checklist

### For Each Page
- [x] Unique title tag (55-60 chars)
- [x] Meta description (150-160 chars)
- [x] H1 tag matches primary keyword
- [x] H2/H3 structure logical
- [x] Primary keyword in first 100 words
- [x] Internal links to other relevant pages
- [x] External links to authoritative sources
- [x] Alt text on all images
- [x] Mobile-responsive design
- [x] Fast loading (< 3s)

## Link Building Strategy

### Internal Linking
- All location pages link to /sell
- All blog posts link to /sell and location pages
- Footer contains all key page links
- Header navigation covers main sections

### External Link Targets
1. **Local Directories**
   - Google Business Profile
   - Yelp
   - Yellow Pages
   - Local chamber of commerce

2. **Industry Directories**
   - Comic shop directories
   - eBay store directories
   - Whatnot seller profiles

3. **Content-Based**
   - Guest posts on comic blogs
   - HARO responses (comic expert)
   - Reddit AMAs in r/comicbookcollecting
